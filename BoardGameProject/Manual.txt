
Gomoku Manual:

1.[Input Format] 
Input format: Enter two numbers between 1 and 10 (inclusive), separated by a space. 
The first number represents the row and the second number represents the column.
"ROW COL": 5 6 means to place the chess piece in the 5th row and the 6th column.
2.[Winning condition] 
Five chesses in a row (horizontally, vertically and diagonally)
3.[Default rules] 
Player1 goes first. 
If it is computer vs. human mode, the default computer is Player1.
4.[Command list]
{Enter "save"}: save the current board information, 
the file save path is C:\Users\{Username}\Documents\MyBoardGameSaves.
{Enter "load"}: restore the board information from the save file, enter the full path information, 
and the game type and game mode of the restored file should be consistent with the current game type and mode.
{Enter "undo"}: enter the number of rounds, and perform the operation of undoing to a certain round (previous).
{Enter "redo"}: the undo operation will be confirmed after the undo operation is completed, 
and entering "redo" will cancel the undo operation.
{Enter "help"}: display this manual again.



Notakto Manual:

[1.Input Format] 
Input format: Enter THREE numbers between 1 and 3 (inclusive), separated by a space. 
The first number represents the board number,
the second number represents the row, and
the third number represents the the column.
"BOARD ROW COL": 1 2 3 means play the chess in the second row and third column of the first board.
[2.Winning condition] 
When a board has three in a row (horizontally, vertically and diagonally), 
the board will no longer be able to play chess. 
When all three boards cannot play chess, 
the player who played chess last loses, and the other player wins.
[3.Default rules] 
Player1 goes first. 
If it is computer vs. human mode, the default computer is Player1.
[4.Command list]
{4.1Enter "save"}: save the current board information, 
the file save path is C:\Users\{Username}\Documents\MyBoardGameSaves.
{4.2Enter "load"}: restore the board information from the save file, enter the full path information, 
and the game type and game mode of the restored file should be consistent with the current game type and mode.
{4.3Enter "undo"}: enter the number of rounds, and perform the operation of undoing to a certain round (previous).
{4.4Enter "redo"}: the undo operation will be confirmed after the undo operation is completed, 
and entering "redo" will cancel the undo operation.
{4.5Enter "help"}: display this manual again.
